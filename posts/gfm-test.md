## 脚注

GitHub Flavored Markdown は、Markdown の拡張仕様です[^1]。remark と rehype を組み合わせることで、強力な Markdown 処理パイプラインを構築できます[^2]。

この記事で使用している remark-gfm プラグインは、unified エコシステムの一部です[^3]。

[^1]: GFM は、GitHub で使用されている Markdown の方言で、テーブルやタスクリストなどの追加機能を提供します。
[^2]: remark は Markdown を処理し、rehype は HTML を処理します。両者を組み合わせることで、Markdown から HTML への変換が可能になります。
[^3]: unified は、構造化テキストを処理するためのインターフェースを提供するプロジェクトです。

## まとめ

remark-gfm プラグインを追加することで、以下の機能が使えるようになりました:

1. ✅ **テーブル** - データを整理して表示
2. ✅ **タスクリスト** - チェックボックス付きリスト
3. ✅ **取り消し線** - テキストに取り消し線
4. ✅ **脚注** - 詳細情報を脚注として追加
5. ✅ **自動リンク** - URL の自動リンク化

これらの機能により、より表現力豊かな Markdown 記事を作成できます!
